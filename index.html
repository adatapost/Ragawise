    • Ragawise

    A real-time raga recognition (prototype) system for web browsers

    Project maintained by sankalpg Hosted on GitHub Pages — Theme by mattgraham

    Dependencies

    • Flask, Flask-CORS
    • JQuery, D3.js

    Web interface (Beta)

    RAGAWISE (BETA): TRY IT YOURSELF

    How to use the web-interface

    • Select your tonic (the base pitch, which is considered as 'Sa')
    • Hit the start recording button, and sing a raga !!!

    For best results use Chrome-stable or Firefox (latest version).
